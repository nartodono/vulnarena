# Vulnarena Dynamic Lab

This directory contains the source code for Vulnarena's dynamic lab.  
**Warning:** The entire lab is intentionally designed to be vulnerable for educational and learning purposes only. **Do not use this as an example for production-level applications.**

## Included Vulnerabilities

This lab features seven hands-on web vulnerabilities:

- **Cross-Site Scripting (XSS)**
- **SQL Injection**
- **Insecure Direct Object References (IDOR)**
- **Broken Access Control**
- **Business Logic Vulnerability**
- **Command Injection**
- **Path Traversal**

## Technology Stack

- **Backend:** PHP (native)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MySQL
