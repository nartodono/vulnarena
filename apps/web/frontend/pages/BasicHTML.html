Pelajari dasar-dasar HTML, bahasa markup utama yang digunakan untuk membuat dan menampilkan halaman web. Pahami bagaimana elemen-elemen HTML bekerja untuk membangun struktur dan konten web yang dinamis dan informatif.

Apa itu HTML?

HTML (HyperText Markup Language) adalah fondasi dari setiap halaman web. HTML menggunakan tag untuk memberi tahu browser bagaimana menampilkan teks, gambar, tautan, dan berbagai elemen lainnya. HTML tidak bersifat fungsional seperti JavaScript, tapi sangat penting untuk struktur.

Struktur Dasar HTML

File HTML memiliki kerangka standar yang dimulai dengan deklarasi <!DOCTYPE html> dan elemen <html>. Contoh sederhana:

        <!DOCTYPE html>
        <html lang="en">
        <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Halaman Saya</title>
        </head>
        <body>
          <h1>Selamat Datang!</h1>
          <p>Ini adalah paragraf pertama saya.</p>
        </body>
        </html>

Tag Dasar yang Wajib Diketahui

  • <h1> - <h6>: Heading, dari yang paling besar (h1) ke paling kecil (h6).
  • <p>: Untuk paragraf teks biasa.
  • <a>: Link ke halaman lain atau URL eksternal.
  • <img>: Untuk menyisipkan gambar.
  • <ul>, <ol>, <li>: Membuat list tak berurut dan berurut.
  • <div>: Container fleksibel untuk layout dan styling.

Contoh Penggunaan Tag

Contoh penggunaan tag untuk membuat konten halaman sederhana:

        <h1>Judul Halaman</h1>
        <p>Ini paragraf pertama. <a href="https://example.com">Klik di sini</a> untuk belajar lebih lanjut.</p>
        <ul>
          <li>Item 1</li>
          <li>Item 2</li>
        </ul>
        <img src="gambar.jpg" alt="Deskripsi Gambar">

Mengapa Belajar HTML Penting?

HTML adalah dasar dari semua halaman web. Pemahaman yang baik akan HTML membantu kamu membuat website yang terstruktur dengan baik, ramah SEO, dan memudahkan pengelolaan konten. HTML juga menjadi fondasi yang harus dikuasai sebelum mempelajari CSS dan JavaScript.
